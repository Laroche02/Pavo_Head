# Pavo Head

A fully parametric OpenSCAD analog conversion canopy for the BetaFPV Pavo Pico II.

Features

- Caddx Ant support
- Rush Tiny Tank support
- ViFly Finder Mini support
- TPU optimized
- Adjustable camera angle
- Print-in-place cable routing
- JST-SH modular wiring
